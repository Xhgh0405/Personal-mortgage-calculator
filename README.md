# 個人房貸試算器

## 專案介紹
這次做的是一個用 C# Windows Forms 寫的個人房貸試算器。  
使用者可以輸入房屋總價、自備款、年利率、貸款年限，還有寬限期，程式會幫忙算出貸款總金額、每月應繳金額、首期利息、首期本金、總利息支出跟總還款金額。

我做這個程式的主要目的是把課堂上學到的 GUI 設計、事件處理、輸入驗證跟公式計算整合在一起，做成一個可以實際操作的小工具。

---

## 功能說明
這個程式目前有以下功能：

- 可以輸入房屋總價
- 自備款可以選擇用「比例」或「金額」輸入
- 可以輸入年利率
- 可以輸入貸款年限
- 可以輸入寬限期
- 按下按鈕後可以自動計算：
  - 貸款總金額
  - 每月應繳金額
  - 寬限期每月應繳金額（只繳利息）
  - 首期利息
  - 首期本金
  - 總利息支出
  - 總還款金額
- 結果會自動顯示成金額格式（有千分位跟小數點後兩位）
- 如果輸入錯誤，會跳出提示訊息，不會直接當掉

## 使用方式
1. 開啟 Visual Studio，載入這個專案。  
2. 按下執行。  
3. 在左邊輸入：
   - 房屋總價
   - 自備款比例或自備款金額
   - 年利率
   - 貸款年限
   - 寬限期
4. 按下 **開始計算**。  
5. 右邊就會顯示試算結果。

---

## 結果說明
按下「開始計算」後，右邊會顯示以下內容：

- **貸款總金額**：實際向銀行申請貸款的金額  
- **每月應繳金額（本 + 息）**：每個月平均要繳的金額  
- **寬限期每月應繳（僅利息）**：如果有設定寬限期，會顯示寬限期內每月只需要繳的利息  
- **首期利息**：第一期付款中利息的部分  
- **首期本金**：第一期付款中本金的部分  
- **總利息支出**：整筆貸款在還款期間總共多付出的利息  
- **總還款金額**：本金加上全部利息後，最後總共要還的金額  

程式顯示的金額都會自動加上千分位，並保留到小數點後兩位，比較方便閱讀。

---

## 公式說明

### 1. 貸款總金額
貸款總金額 = 房屋總價 - 自備款

### 2. 月利率
月利率 = 年利率 ÷ 12

如果年利率輸入的是百分比，程式會先轉成小數再計算。  
例如年利率 2.15%，實際計算時會先變成 0.0215。

### 3. 每月應繳金額（本息平均攤還）
如果沒有寬限期，或寬限期結束後，房貸會用本息平均攤還公式計算：

```text
M = P × [ r(1+r)^n ] / [ (1+r)^n - 1 ]
```
