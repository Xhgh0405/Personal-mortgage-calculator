# 個人房貸試算器

## 專案簡介
本專案是一個使用 C# Windows Forms 製作的「個人房貸試算器」，提供使用者輸入房屋總價、自備款、年利率、貸款年限與寬限期，快速計算出房貸相關結果，包含貸款總金額、每月應繳金額、首期利息、首期本金、總利息支出與總還款金額。  
本程式介面以簡單、清楚、易操作為設計目標，並加入輸入驗證機制，避免因錯誤資料造成程式異常。

---

## 功能特色
- 可輸入房屋總價
- 可選擇以 **自備款比例（%）** 或 **自備款金額（元）** 方式輸入
- 可輸入年利率（%）
- 可輸入貸款年限（年）
- 可輸入寬限期（年）
- 自動計算：
  - 貸款總金額
  - 每月應繳金額（本息攤還）
  - 寬限期每月應繳金額（僅利息）
  - 首期利息
  - 首期本金
  - 總利息支出
  - 總還款金額
- 結果自動格式化為金額顯示（千分位 + 小數點後兩位）
- 內建輸入驗證，防止不合法資料輸入

## 執行方式
1. 下載或複製本專案到本機。
2. 使用 Visual Studio 開啟專案資料夾或 `.sln` 解決方案檔。
3. 按下 **F5** 或點選「開始執行」。
4. 開啟程式後，在介面中輸入房貸相關資料。
5. 按下 **開始計算** 按鈕，即可看到試算結果。

---

## 使用說明

### 1. 輸入資料
使用者需輸入以下資訊：
- **房屋總價（元）**
- **自備款輸入方式**
  - 比例（%）
  - 金額（元）
- **年利率（%）**
- **貸款年限（年）**
- **寬限期（年，可填 0）**

### 2. 計算結果
按下「開始計算」後，右側會顯示：
- **貸款總金額**
- **每月應繳金額（本 + 息）**
- **寬限期每月應繳（僅利息）**
- **首期利息**
- **首期本金**
- **總利息支出**
- **總還款金額**

## 預設值

- 自備款金額：`2,000,000`
- 年利率：`2.15`
