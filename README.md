# 個人房貸試算器

## 專案介紹
這是一個用 C# Windows Forms 寫的個人房貸試算器。  
使用者可以輸入房屋總價、自備款、年利率、貸款年限，還有寬限期，程式會幫忙算出貸款總金額、每月應繳金額、首期利息、首期本金、總利息支出跟總還款金額。

這個程式的主要目的是把課堂上學到的 GUI 設計、事件處理、輸入驗證跟計算公式整合在一起，做成一個可以實際操作的小工具。

---

## 功能說明
這個程式目前有以下功能：

- 可以輸入房屋總價
- 自備款可以選擇用「比例」或「金額」輸入
- 可以輸入年利率
- 可以輸入貸款年限
- 可以輸入寬限期
- 按下按鈕後可以自動計算：
  - 貸款總金額
  - 每月應繳金額
  - 寬限期每月應繳金額（只繳利息）
  - 首期利息
  - 首期本金
  - 總利息支出
  - 總還款金額
- 結果會自動顯示成金額格式（有千分位跟小數點後兩位）
- 如果輸入錯誤，會跳出提示訊息，不會直接當掉

---

## 使用方式
1. 開啟 Visual Studio，載入這個專案。
2. 按下執行。
3. 在左邊輸入：
   - 房屋總價
   - 自備款比例或自備款金額
   - 年利率
   - 貸款年限
   - 寬限期
4. 按下 **開始計算**。
5. 右邊就會顯示試算結果。
